# Contributing

## Branch Naming
- `feature/<short-name>`
- `fix/<short-name>`
- `chore/<short-name>`

## Pull Requests
- Keep PRs small and focused
- Explain what changed and why
- Link related issue if available

## Commit Messages
Follow Conventional Commits:
- `feat:` new features
- `fix:` bug fixes
- `docs:` documentation changes
- `refactor:` code refactors
- `chore:` maintenance
